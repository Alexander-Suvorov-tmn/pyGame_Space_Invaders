# pyGame_Space_Invaders

## Простая игра Space Invaders с применением ООП

## Стек
- Python > 3.6
- SQLite3

## Лицензия
- MIT


## Get start
- git clone https://github.com/halexx7/DJ_uk_newcity.git
- cd pyGame_Space_Invaders/
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- python3 game.py
